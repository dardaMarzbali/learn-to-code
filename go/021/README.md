# No.021

問題: 簡単なタスクトラッカーアプリケーションを作成しています。タスクにはタイトル（string型）と完了ステータス（bool型）が含まれます。以下のタスクを達成するためのGoプログラムを書いてください。

1. タスクを表す構造体 `Task` を作成してください。この構造体にはタイトル（string型）と完了ステータス（bool型）の2つのフィールドが必要です。
1. タスクを格納するためのスライス `tasks` を作成してください。このスライスには初期のタスクが含まれています。
1. 新しいタスクを追加する関数 `AddTask` を実装してください。この関数はタイトル（string型）を受け取り、新しいタスクを `tasks` スライスに追加します。新しいタスクの完了ステータスはデフォルトで `false` です。
1. タスクの完了ステータスを切り替える関数 `ToggleTask` を実装してください。この関数はタイトル（string型）を受け取り、指定されたタイトルのタスクの完了ステータスを切り替えます（`true` から `false` または `false` から `true`）。
1. タスクを削除する関数 `DeleteTask` を実装してください。この関数はタイトル（string型）を受け取り、指定されたタイトルのタスクを `tasks` スライスから削除します。
1. タスクの一覧を表示する関数 `ListTasks` を実装してください。この関数は `tasks` スライス内の全てのタスクを表示します。タスクが完了しているかどうかも表示するようにしてください。
