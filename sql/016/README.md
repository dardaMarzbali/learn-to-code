# No.016

あなたは映画レビューサイトのデータベース管理者であり、`movies` テーブルから映画を評価の高い順に並べ、上位5つの映画の情報を取得するSQLクエリを作成する必要があります。情報には映画のタイトルと評価が含まれる必要があります。

#### `movies` テーブル:

- `movie_id` (INT): 映画のID
- `title` (VARCHAR): 映画のタイトル
- `rating` (FLOAT): 映画の評価（1.0から10.0のスケール）

#### 要件:

1. 映画を評価が高い順に並べ替えます。
2. 上位5つの映画のタイトルと評価のみを表示します。
